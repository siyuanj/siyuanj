# Repository Workflow

This repository powers Siyuan Jiang's GitHub profile README. GitHub renders `README.md` from this special `siyuanj/siyuanj` repository at the top of the public profile page.

## Read Before Substantial Work

Before non-trivial changes, read:

- `AGENTS.md`
- `README.md`
- `docs/STATUS.md`

## Editing Notes

- Keep `README.md` concise, scannable, and profile-oriented.
- Prefer stable links and public information.
- Use external dynamic SVG services sparingly so the profile stays calm and loads quickly.
- Keep the profile closer to a concise README than a visual dashboard.

## Documentation Updates

After meaningful changes, update `docs/STATUS.md` with what changed, why, current state, and the next recommended step.
